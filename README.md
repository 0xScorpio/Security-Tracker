# Security Tracker

**Security Tracker** is an offline single-page web app for penetration testers to manage engagements. It provides guidance through a proven methodology while tracking what's been done on each target. Features include:

- **Machine management**
- **Methodology checklist**
- **Evidence tracking**
- **Progress dashboard**
- **Timeline view**
- **Local storage persistence**


```bash

git clone https://github.com/0xScorpio/Security-Tracker.git
```

And then run a local HTTP server to run it:

```bash

python -m http.server 8000

```

Access the server and happy hacking!

```bash

http://localhost:8000
```
